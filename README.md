# DİSCORD QUEST CODE İLE GÖREV KASMAK

Discord Orbs görevlerini oynamadan kasmak için  **Quest Code** İle Kasabilirsiniz bu sayede mağaza istediğiniz ürünleri kullanabilirsiniz. 

> 

---

## Hızlı bakış

| Bileşen | Port | Görev |
|---|---|---|
| `settings.json` | Konsol Yerini Açmanıza Yarar | Ctrl+Shift+I |
| `Quest Code` | Görevleri Yapmak İçin Konsol Kodu | • |


---

## Konsol Panelini Açma

### Settings.json

Bilgisiyarınızdan Çalıştırı Açıp **AppData** Klasörünü Aratınız 
**AppData** Klasörünün İçinde **Roaming** Klasörüne Girin ve Ardından **discord** Adlı dosyayı açın 
devamında içindeki **settings.json** dosyasını silip size verdiğim **settings.json** dosyasını atın ve kapatın. 

## Discord Kurulum
### Discord Açıp Görev Mağazasını açın.
Görevler sizde gözükmüyor size vpn uygulaması ile discordu açıp tekrar deneyin veya vpn lokansyonu veya vpn programını değiştirin.
İstediğiniz görev kısmına gelip göreve başlayarak Masaüstü ile Göreve başlama seçeneğine tıklayın. 
ve ardından **Ctrl+Shift+I** Konsol komutunu açın Yukarıdan **Console** Sekmesine basın
İndirmiş olduğununuz `Quest Code.txt` metin belgesinin içindeki kodu kodu kopyalayın ve **Console** Yerine Yapıştırın. 
Hata verir ise yapıştıramazsanız 

`allow pasting`

`"allow pasting"`

`'allow pasting'`

Bu Üç Kodu aratarak Ctrl+v Özelliğini Açın ve kodu yapıştırın.
